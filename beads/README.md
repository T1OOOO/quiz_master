# Beads - Agentic Memory System

Beads - это система памяти для AI-агентов, позволяющая сохранять и извлекать информацию о задачах, решениях и контексте проекта.

## Установка

Система уже инициализирована в проекте. Для использования выполните:

```bash
beads init
```

## Использование

### Создание задачи

```bash
beads create "Описание задачи"
```

### Просмотр задач

```bash
beads list
```

### Поиск задач

```bash
beads search "ключевое слово"
```

## Структура

- `beads/` - основная директория системы
- `beads/memory/` - хранилище памяти агента
- `beads/tasks/` - задачи агента
- `beads/config.json` - конфигурация системы
