## 📦 Загрузка Ассетов

### Изображения вопросов
**Загружаются с сервера** ✅
- Изображения вопросов (`question.image_url`) загружаются через `Image.network()`
- URL формируется через `ImageUtils.getImageUrl()` для поддержки относительных путей
- Добавлен `loadingBuilder` для показа индикатора загрузки
- Добавлен `errorBuilder` для обработки ошибок

### Фоновые изображения
**Могут быть локальными или с сервера** ⚠️
- Фоновые изображения для тем (например, `home-alone-bg.jpg`) могут быть:
  - Локальными (через `AssetImage`) - для holiday темы
  - С сервера (через `Image.network`) - если указан полный URL

### Текстуры и декоративные элементы
**Локальные** (как в React Native)
- Текстуры для карточек (`bg_soft_premium.png`) - локальные
- Иллюстрации (`bg_quiz_illustrated.png`) - локальные
- Иконки категорий - встроенные Material Icons

### Рекомендации

#### Для вопросов
```dart
// Изображения вопросов всегда с сервера
Image.network(
  ImageUtils.getImageUrl(question.imageUrl),
  fit: BoxFit.cover,
)
```

#### Для фонов
```dart
// Может быть локальным или с сервера
themeConfig.backgroundImage != null
  ? Image.network(themeConfig.backgroundImage!) // Если URL
  : AssetImage(themeConfig.backgroundImage!)     // Если локальный путь
```

## 🔧 Утилиты

### ImageUtils
Создан класс `ImageUtils` для работы с изображениями:
- `getImageUrl()` - формирует полный URL для изображения
- `isNetworkImage()` - проверяет, является ли изображение сетевым
